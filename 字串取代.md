## Java字串取代
### StringBuffer
#### replace()方法
public StringBuffer replace(int start, int end, String str)
第一個參數 start : 起始索引值(包含)
第二個參數 end   : 結束索引值(不含)
第三個參數 str   : 為一字串，用於取代從start至end-1的字串
  
  StringBuffer buffer = new StringBuffer("15197447018");
  buffer.replace(3,7,"****");
  System.out.println(buffer.toString());
